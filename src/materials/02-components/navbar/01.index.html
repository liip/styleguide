--- index: true notes: | ##### Modifiers
.navbar--negative Use light text color
.navbar--floating Absolutely position it above the rest
.navbar--sticky Keep it always visible at the top of the screen
.navbar--overlay Adds a slightly transparent light background
##### Make it shrink ! In case you use the sticky navbar, you should make it shrink while the user scroll to free-up some space. To do so, use the following code: ```javascript window.addEventListener('DOMContentLoaded', function() { var navbar = document.querySelector('#my-sticky-navbar'); new Toolkit.Navbar(navbar, { mode: 'window' }); }); ``` There is two modes available: `window` and `instant`, the latter being by default. `instant` make it shrink as soon as the user scroll. `window` start the transition when the end of the first screen approach the top, corresponding to the beginning of the second fitted section. You can see example of implementation in the example pages. ---
