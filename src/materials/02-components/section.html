--- notes: | Useful to structure a page. The content is automatically spaced from the sides. By default it has no background but you are free to add a `.section__bg` child which will cover the whole space. Ensure the contast with the text is good enough according to the image chosen. ##### Modifiers
.section--fit Take at least the viewport height
.section--reversed Use light text color
.section--primary Green gradient background and light text color
.section--dark Dark solid background and light text color
---

Default section

  1. Height depending on content
  2. No background
  3. Normal text color

Light background section

  1. Height depending on content
  2. Light background image
  3. Normal text color

Dark background fitted section

  1. Full viewport height
  2. Dark background image
  3. Light text color

Primary section

  1. Height depending on content
  2. Green gradient background
  3. Light text color

Dark section

  1. Height depending on content
  2. Dark warm grey background
  3. Light text color
