--- notes: | Use [OpenWeatherMap API](http://openweathermap.org/API) to display current weather at a defined city. Below is the current weather in Bern, Switzerland. A `data-city` attribute containing the City ID is required on the element. You can find a city ID on [openweathermap.org/city](http://openweathermap.org/city). ##### Initialization ```javascript window.addEventListener('DOMContentLoaded', function() { var el = document.querySelector('#my-weather'); var weather = new Toolkit.Weather("YOUR-OPEN-WEATHER-MAP-API-TOKEN"); weather.init(el); }); ``` ##### Methods
weather.init(DOMElement) Init for a single item
weather.initAll() Init all .weather elements in the page
